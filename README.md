## About
Pulse-X is a Wi-Fi powered, cylindrical shaped, JBL inspired speaker engineered to dominate high noise outdoor environments. Built to be completely independent, it bypasses traditional Bluetooth limitations and cell data reliance by creating its own local Wi-Fi Access Point.

Driven by a high-performance dual-core ESP32-S3 processor and backed by 8MB of PSRAM, Pulse-X delivers lag-free, lossless Hi-Fi audio streaming directly to an integrated I2S DAC and a roaring amplifier system. Complete with an on-board OLED dashboard for real-time track monitoring and tactile button controls, it is a rugged, standalone audio powerhouse built to bring the ultimate wireless sound experience straight to the heart of the stadium crowd.
